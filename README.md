# SarahTwemlow-TechTest

## Prerequisites

- node version: `v18.13.0`
- npm version: `8.19.3`

## Install

run `npm install`

## Running the Tests

- run `npx cypress open`
- select E2E Testing
- select Start E2E Testing in Chrome
- select Agify.feature

Alternatively, to run the tests headlessly run:
`npx cypress run`
